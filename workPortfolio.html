Giphy API Fun

This is a basic project that leverages the Giphy API to return a random number of gifs related to a search term. You can return gifs from the buttons provided or you can create your own button and return the gifs. The gifs are static, but when you click on the image, they animate.

Train Schedule

This project is a demo using Firebase realtime database and moment.js. It's a fairly simple example. When the page loads, the database loads previously added trains. New trains can be added using the form at the bottom of the page. Moment.js is used to calculate the next train's arrival time.

YumYumNav

YumYumNav is an example of a JavaScript project that I worked on. It's a simple restaurant finding app that uses the OpenTable API and the Google Maps API, along with Firebase and Bootstrap. It allows the user to enter a city or a zipcode and search for a restaurant. Try it!

Crystal Collector

The Crysal Game is a simple JavaScript game. It uses Bootstrap and jquery to create interactive elements on the page. The user can click on one of the crystals to try to match the randomly generated number. Each crysal's value has a random number generated behind it at the start of the game. See if you can match the number!
